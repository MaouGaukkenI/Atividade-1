Arrumando código de conexão ao banco de dados:

$ git add .
//Carrego as modificações no meu arquivo

$ git status
//Verifico se todas modificações foram carregadas

$ git commit -m "Banco de dados login v1.0.0"
//Salvo um commit com as modificações necessárias para integrar meu sistema ao banco de dados

$ git status
//Verifico se foram salvas com sucesso

$ git push
//Envio para o GitHub para salvar as modificações a nuvem
